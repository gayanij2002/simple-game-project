# Simple Game Project

This is a simple browser-based game developed using HTML, CSS, and JavaScript. The project was created to practice front-end web development skills and implement basic interactive game logic using core web technologies.

## Technologies Used

* HTML
* CSS
* JavaScript

## Project Demo Video

Watch the project demo here:
https://youtu.be/nPIRJMncMXQ

## Features

* Interactive gameplay
* User-friendly interface
* Responsive design
* Basic game logic implementation

This project demonstrates my understanding of front-end web development and my ability to create simple interactive applications for the browser.
